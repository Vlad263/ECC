# Executive Command Center (ECC)

Central AI system for productivity, multi-agent automation, and execution intelligence.

## Core Goals
- Prior-setting engine (daily control system)
- Learning curator
- Networking assistant
- Later: business automation modules

## Tech stack 
- Python
- Google ADK / MCP / Make integrations
- Vector memory + sheet-based models


## Structure 
agents/ -> Agent logic + behaviours
core/ -> Services, utilities, config handling
pipelines/ -> Routines connecting agents
models/ -> embeddings + fine-tuned models
docs/ -> architecture, research, development notes
tests/ -> automated tests
